# Loan Application React Project

A simple loan application form built using React.
This project is part of my React learning roadmap and was creating to practice core React concepts such as components, state management, routing, context, and form handling.

---

## Features

- User input form for:
  - Name
  - Email
  - Phone number
- Student status selection using a checkbox
- Salary range selection using radio buttons
- Alerts for user feedback
- Basic form validation

---

## Concepts Practiced

- React functional components
- `useState` for managing form data
- `useContext` for shared/global state
- React Router for navigation between pages
- Controlled form inputs:
  - Text inputs
  - Checkbox
  - Radio buttons
  - Select Dropdown
- Conditional rendering
- User alerts and feedback
